# XAI for Coding Agent Failures: Transforming Raw Execution Traces into Actionable Insights

## 基本信息

- **年份**: 2026
- **arXiv ID**: 2603.05941
- **作者**: Arun Joshi

## 核心贡献

该研究提出了一个系统的可解释AI框架，旨在将原始Agent执行trace转化为结构化、可解释的解释。该方法包括三个组件：领域特定的失败分类法、自动注释系统和混合解释生成器（产生可视化流程和自然语言洞察）。用户研究表明，该方法使"用户识别失败根因的速度提高了2.8倍，提出正确修复建议的准确率提高了73%"。

## 科技界方法

1. **领域特定失败分类法**: 针对编程Agent定义专门的失败类型
2. **自动注释系统**: 自动标记trace中的关键事件
3. **混合解释生成器**: 结合可视化流程和自然语言解释
4. **可解释AI (XAI)**: 提供人类可理解的诊断解释

## 实验结果

- 用户识别根因速度提升2.8倍
- 提出正确修复建议的准确率提高73%
